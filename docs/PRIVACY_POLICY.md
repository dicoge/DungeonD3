## 中文版

### 1. 資訊收集
DungeonD3（以下簡稱「本應用程式」）**不會收集、儲存或傳輸**任何個人識別資訊。

本應用程式使用以下本地儲存：
- **AsyncStorage**：僅用於儲存遊戲存檔資料（玩家進度、道具、設定），所有資料僅儲存在您的裝置本地，不會上傳至任何伺服器。

### 2. 網路使用
- 本應用程式為**完全離線**遊戲，不需要網路連線即可遊玩。
- 本應用程式**不會**連線至任何第三方伺服器、分析服務或廣告網路。
- 本應用程式**不會**收集使用統計資料、裝置資訊或位置資料。

### 3. 第三方服務
- 本應用程式**不使用**任何第三方分析、廣告或社交媒體 SDK。
- 本應用程式**不包含**任何應用內購買（IAP）或外部連結。

### 4. 資料安全
由於本應用程式不收集任何個人資料，因此不存在資料外洩風險。所有遊戲資料儲存在您的裝置本地，由作業系統的應用程式沙箱保護。

### 5. 兒童隱私
本應用程式適合所有年齡層（年齡分級：4+ / Everyone），但不會收集任何兒童的個人資訊。

### 6. 政策更新
本隱私政策可能不定期更新。重大變更將透過應用程式更新通知。

## English Version

### 1. Information Collection
DungeonD3 (hereinafter referred to as "the Application") **does not collect, store, or transmit** any personally identifiable information.

The Application uses the following local storage:
- **AsyncStorage**: Used solely for storing game save data (player progress, items, settings). All data is stored locally on your device and is never uploaded to any server.

### 2. Network Usage
- The Application is a **fully offline** game. No internet connection is required to play.
- The Application **does not** connect to any third-party servers, analytics services, or advertising networks.
- The Application **does not** collect usage statistics, device information, or location data.

### 3. Third-Party Services
- The Application **does not use** any third-party analytics, advertising, or social media SDKs.
- The Application **does not contain** any in-app purchases (IAP) or external links.

### 4. Data Security
Since the Application does not collect any personal data, there is no risk of data breach. All game data is stored locally on your device, protected by the operating system's application sandbox.

### 5. Children's Privacy
The Application is suitable for all ages (Rating: 4+ / Everyone) and does not collect any personal information from children.

### 6. Policy Updates
This privacy policy may be updated from time to time. Significant changes will be notified through application updates.
