# DungeonD3 — 遊戲設計規格書

## 1. 核心玩法循環（每回合）

```
[擲D20] → [根據點數規劃行動] → [移動/攻擊/道具/待機] → [敵人行動] → [回合結束] → 回到[擲D20]
```

### 行動點數系統
| D20 擲出 | 獲得行動點 |
|---|---|
| 1~6 | 1 點 |
| 7~13 | 2 點 |
| 14~20 | 3 點 |

### 行動消耗
| 行動 | 消耗 | 結束回合？ |
|---|---|---|
| 移動 1 格 | 1 點 | 否（可連續移動） |
| 攻擊敵人 | 1 點 | **是，攻擊後立刻結束回合** |
| 使用道具 | 2 點 | 是 |
| 待機/結束 | 0 點 | 是 |
| 行動點用盡 | — | 自動結束回合 |

## 2. 教學關卡（floor 0）

### 地圖設計（12×10 格）
```
############
#..........#
#..S.......#  S = 玩家 (3,2)
#..........#
#..E..I...#  E = 敵人 (3,4), I = 道具 (6,4)
#..........#
#..........#
#..........#
#........P.#  P = 階梯 (9,8)
############
```

**教學流程（7 步驟）：**

| 步驟 | ID | 觸發條件 | 提示內容 | UI 呈現 |
|---|---|---|---|---|
| 1 | 場景認知 | 遊戲開始 | 地圖元素說明 | Modal overlay |
| 2 | 骰子說明 | modal 按鈕 | D20 規則 | 骰子頁 |
| 3 | 移動教學 | 骰完後自動 | 行動點說明 + 走向敵人 | HUD hint |
| 4 | 攻擊教學 | 移動後自動 | ⚔️ 攻擊說明 + 攻擊後結束回合 | HUD hint |
| 5 | 道具教學 | 撿道具後 | 道具使用說明 | HUD hint |
| 6 | 結束回合 | 點結束按鈕 | 待機節省行動力 | HUD hint |
| 7 | 通關 | 走向階梯 | 進入第一層 | 自動觸發 |

**重要設計原則：**
- 玩家位於 (3,2)，敵人位於 (3,4) — 相鄰格距離為 2
- 擲出 2+ 點時：1 步移動（pts-1）+ 1 次攻擊（pts-1 → 0 → 攻擊後結束）
- 擲出 1 點時：1 步移動（pts-1 → 0 → 自動結束回合）→ 下回合攻擊
- **教學關卡敵人 HP=5，玩家 ATK=5（保證 1 擊斃命）**

## 3. 地城結構（5層 + BOSS）

| 樓層 | 名稱 | 敵人難度 | 道具稀有度 |
|---|---|---|---|
| 1 | 地城入口 | 普通 | Common |
| 2 | 遺忘墓穴 | 普通+ | Uncommon |
| 3 | 惡魔深淵 | 困難 | Rare |
| 4 | 遠古工場 | 困難+ | Rare |
| 5 | 暗影國度 | BOSS 層 | Epic |

### BOSS（巫妖王）
- HP: 180
- 攻擊力: 12
- 技能: 骨矛風暴（15傷）、靈魂收割（22傷）
- 擊敗後遊戲勝利

---

## 4. 失敗條件
- HP <= 0 → 遊戲結束
- 顯示死亡畫面，可選重新開始

## 5. 視覺元素需求

### 瓷磚視覺
| 類型 | 背景色 | 效果 |
|---|---|---|
| 地板 | #1a1a30 | — |
| 牆壁 | linear-gradient(135deg,#2a2a45,#1a1a30) | inset shadow |
| 玩家 | #00e5ff | box-shadow glow + pulse 動畫 |
| 敵人 | #ef5350 | box-shadow 敵意感 |
| 道具 | #ffe500 | itemGlow 動畫（1.5s 脈動）|
| 階梯 | #00c853 | box-shadow 綠光 |
| 陷阱 | #ff6600 | box-shadow 危險警告 |
| BOSS | #8b0000 | box-shadow 紅光 + bossGlow 動畫 |

### D20 骰子
- Canvas 160×160px
- 六邊形外框 + 兩圈三角形 + 中央數字
- 紫色主題 (#6a0dad → #9b30d0)
- 1.5 秒滾動動畫，最後 0.3 秒 Settle
