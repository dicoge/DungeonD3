> 版本：1.1 | 目標：上架 iOS/Android App Store + 網頁

## 1. 遊戲核心體驗

### 1.1 一句話描述
**「用 D20 骰子決定命運的地城爬塔 RPG」**

玩家在自動生成的地城迷宮中探索，透過 D20 骰子戰鬥系統與敵人戰鬥，購買道具強化自己，挑戰每 5 層的 Boss，最終通過 15 層。

### 1.2 目標用戶
- 休閒玩家（15-30 分鐘一局）
- D&D/桌遊愛好者
- RPG 爬塔類型粉絲

---

## 2. 遊戲核心循環

```
[進入地城] → [新手教學] → [第1層開始]
                              ↓
                    ┌─────────┴─────────┐
                 [擲骰子]          [擲骰子]
                    ↓                 ↓
              [獲得移動點數]    [戰鬥結算]
（MOVE點數 = 骰值）    （D20 ≥ 敵人DEF = 命中）
                    ↓                 ↓
              [地圖移動] ← ← ← → [敵人受傷/玩家受傷]
                    ↓
              [遇到事件]
         ┌────┬────┴────┬────┐
       道具  敵人  商店  樓梯
         ↓     ↓         ↓     ↓
      拾取  戰鬥  進入  下一層
                    ↓
              [重複直到15層或死亡]
                    ↓
               [結算畫面]
```

## 3. 核心系統

### 3.1 D20 骰子系統

**回合流程：**
- 玩家擲骰 → 獲得 MOVE 點數 → 消耗行動點移動 → 行動點歸零 → 下一回合可再骰
- 禁止在行動點 > 0 時重複骰
- 玩家永遠先手

**移動模式：**
- 擲骰子 → 獲得 MOVE 點數（MOVE = 骰值 ÷ 3，向下取整，最少 1 點）
- 每移動 1 格消耗 1 MOVE
- MOVE 用完後自動結束回合，擲骰按鈕恢復可用

**戰鬥模式：**
- 玩家攻擊：D20 + ATK vs 敵人 DEF
  - D20 = 1 → Miss（0 傷害）
  - D20 = 20 → 暴擊（2x 伤害 + 額外 D6）
  - 其他：D20 + ATK > DEF = 命中，否則 Miss
- 命中傷害 = ATK + D6
- 暴擊傷害 = ATK + (D6 × 2)

**特殊規則：**
- 玩家永遠先手
- 敵人死亡後才能移動到下一格

### 3.2 地城生成系統

- 每層：8-12 個房間 + 走廊連接
- 房間類型：普通房間、道具房、商店房、Boss房（每5層）
- 自動牆壁生成（房間周圍）
- 樓梯恆定在最遠房間

### 3.3 敵人系統

| 樓層 | 敵人數量 | 敵人強度 |
|------|---------|---------|
| 1-3  | 3-5     | 基礎     |
| 4-6  | 4-6     | +30%    |
| 7-9  | 5-7     | +60%    |
| 10-14| 6-8     | +100%   |
| 15   | Boss    | 3x HP   |

### 3.4 道具系統

**消耗品：** 使用後立即生效
- 生命藥水：恢復 30 HP（$15）
- 炸彈：對敵人造成 40 傷害（$30）
- 鳳凰羽毛：死亡後自動復活（$200）

**裝備：** 永久加成，裝備後生效
- 武器：+ATK
- 護甲：+DEF

### 3.5 商店系統

- 每 3 層出現商店
- 只能用金幣購買
- 金幣來源：擊敗敵人、過關獎勵

## 4. UI/UX 設計

### 4.1 螢幕結構

```
┌─────────────────────────────────┐
│  頂部狀態列（永遠固定）          │
│  [第N層] [❤️HP] [💰金幣] [Lv]   │
├─────────────────────────────────┤
│                                 │
│         主遊戲區域              │
│    （根據當前Tab顯示不同內容）    │
│                                 │
├─────────────────────────────────┤
│  🎲骰子  🗺️地圖  ⚔️戰鬥  🛒背包 │
└─────────────────────────────────┘
```

### 4.2 五個主要畫面

#### 🎲 骰子頁（首頁）
- 大型 D20 骰子視覺效果（SVG 動畫）
- 顯示擲骰結果
- MOVE 點數提示
- 「擲骰」按鈕（行動點 > 0 或已骰過時禁用）
- 當前狀態指引文字

#### 🗺️ 地圖頁
- 地城迷宮（字符地圖，38px 大格子）
- 玩家位置（🧙）與終點（🚪）
- **點擊相鄰格子移動**（上下左右一步）
- MOVE 剩餘點數顯示
- 已探索/未探索區域標記
- 敵人（👾）和道具（💎）位置標記

#### ⚔️ 戰鬥頁
- 敵人資訊卡（名稱、HP條、屬性）
- 攻擊按鈕
- 擲骰結果顯示
- 戰鬥日誌

#### 🛒 商店頁
- 商品列表（名稱、效果、價格）
- 玩家金幣顯示
- 購買按鈕
- 商品庫存

#### 🎒 背包頁
- 道具列表（名稱、數量、效果）
- 裝備區（武器、護甲）
- 使用/卸載按鈕
- 角色屬性總覽

### 4.3 新手教學（Tutorial）

**5 個步驟的強制引導：**

1. **歡迎**
   - 「歡迎來到地城！我是你的嚮導 🧙‍♂️」
   - 「跟著我一步步學習，成為真正的冒險者！」

2. **認識骰子**
   - 「這是 D20 骰子，是命運的象徵！」
   - 「按下去擲骰子，你的冒險從這裡開始！」
   - → 要求玩家按一次骰子

3. **認識地圖**
   - 「地圖會顯示整個地城的結構」
   - 「用方向鍵移動你的角色」
   - 「每走一步都會消耗行動點數」
   - → 要求玩家移動到附近房間

4. **戰鬥教學**
   - 「遭遇敵人了！按攻擊按鈕戰鬥」
   - 「D20 = 1 是 Miss，D20 = 20 是暴擊！」
   - → 要求玩家擊敗第一個敵人

5. **開始冒險**
   - 「恭喜你完成新手教學！」
   - 「記得：每 5 層有 Boss，準備好了嗎？」
   - → 進入正式遊戲

## 6. 存檔設計

### 6.1 存檔內容
```typescript
interface SaveData {
  version: string;
  player: Player;
  floor: number;
  map: DungeonMap;
  enemies: Enemy[];
  items: Item[];
  timestamp: number;
}
```

### 6.2 存檔時機
- 每次進入新樓層
- 每次購買道具
- 每次使用道具
- 每次戰鬥結束
