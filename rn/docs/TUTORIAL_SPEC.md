> 版本：1.0 | 系統：互動引導式新手教學
> 最後更新：2026-05-14
> 對應程式碼：TutorialOverlay.tsx, gameStore.ts (Tutorial Steps)

---

## 設計原則

| 原則 | 說明 | 解決的問題 |
|:----:|------|:----------:|
| **操作引導取代文字說明** | 教學告訴玩家「下一步做什麼」，而非「遊戲規則是什麼」 | ❌「不應該教我」 |
| **互動引導取代被動閱讀** | 玩家必須實際操作才能推進，不能只按「下一頁」 | ❌「怎麼只有文字說明」 |
| **一步一引導** | 每一步只教一個動作，完成後自動進入下一步 | ❌「我希望一步一步引導」 |
| **自動完成關閉** | 教學完成後自動關閉，無需手動按「跳過」 | ❌「最後還得按跳過教學才能關閉」 |

## 步驟流程

```
[歡迎畫面] → [引導擲骰] → [引導移動] → [引導攻擊] → [引導使用道具] → [完成自動關閉]
   步驟 1       步驟 2        步驟 3        步驟 4          步驟 5          步驟 6
```

### 步驟 1：歡迎畫面
- 全螢幕遮罩，中央顯示歡迎卡片
- 卡片內容：🧙 +「歡迎來到地城！」+ 引導文字
- 玩家點擊「🚀 開始冒險！」按鈕進入下一步
- 點擊遮罩無反應（必須按按鈕）

### 步驟 2：引導擲骰
- 自動切到 🎲 骰子 Tab
- 骰子按鈕高亮（紫色脈動光暈）
- 浮動提示卡：「試試看！點擊骰子按鈕擲骰！」
- 監聽 `diceValue !== null` → 自動下一步
- 點擊其他區域無反應

### 步驟 3：引導移動
- 自動切到 🗺️ 地圖 Tab
- 與玩家相鄰的可移動格子（floor）發出藍色光芒
- 浮動提示卡：「點擊發光的格子移動！」
- 監聽 `player.x/y` 變化 → 自動下一步

### 步驟 4：引導攻擊
- 系統自動生成一隻史萊姆在玩家相鄰格
- 敵人格子紅色高亮 → 點擊後進入 BattleOverlay
- 攻擊按鈕紅光脈動高亮
- 監聽 `attack()` 執行 → 自動下一步

### 步驟 5：引導使用道具
- 戰鬥結束後，背包按鈕發出紫色光芒
- 引導打開背包 → 使用生命藥水
- 監聽 `player.hp` 變化 → 自動下一步

### 步驟 6：完成動畫
- 全螢幕 🎉 恭喜動畫
- 2 秒後自動關閉（不用按任何按鈕）

## Zero Skip 原則
- 無「跳過教學」按鈕
- 無「✕ 關閉」按鈕
- 完成後自動關閉
- 強制完成每一步
